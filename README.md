## 📌 프로젝트 개요

ProctorAI는 온라인 시험 환경에서 발생할 수 있는 부정행위를 AI가 자동으로 감지·기록하고, 관리자가 실시간으로 개입할 수 있는 통합 감독 시스템입니다.

MediaPipe 얼굴 추적, Groq LLM, Web Speech API를 결합하여 시선 이탈, 화면 전환, 의심 발언을 실시간 감지하며, 이상 감지 시 AI 챗봇이 자동으로 응시자와 면담을 진행합니다.

## ✨ 주요 기능

### 🎓 응시자 기능

| 기능 | 설명 |
|------|------|
| **시선 추적** | MediaPipe FaceMesh로 홍채 위치를 실시간 분석, 화면 이탈 감지 |
| **얼굴 미감지** | 카메라에서 얼굴이 사라지면 즉시 경고 발생 |
| **화면 이탈 감지** | 다른 탭/창으로 전환 시 즉시 경고 |
| **음성 모니터링** | Web Speech API로 음성을 실시간 기록, Groq AI가 부정행위 여부 분석 |
| **AI 면담** | 경고 2회마다 Groq LLM 기반 AI 감독관과 자동 면담 진행 |
| **경고 시스템** | 누적 6회 경고 시 0점 강제 종료 |
| **수식 렌더링** | LaTeX 수식(`$...$`) 자동 렌더링 (KaTeX) |

### 🖥️ 관리자 기능

| 기능 | 설명 |
|------|------|
| **시험 생성** | 제목·시간·문항 수 설정, 활성/비활성 관리 |
| **AI 문제 생성** | PDF 업로드 → Vision API로 이미지 분석 → 객관식 문제 자동 생성 |
| **실시간 감독** | 응시 중인 학생 현황 실시간 조회 (5초 폴링) |
| **관리자 개입** | 특정 응시자에게 메시지 전송, 시험 일시정지, 강제 종료 |
| **상세 결과 조회** | 답안·이벤트 타임라인·음성기록·AI면담 내역 탭별 확인 |
| **이벤트 타임라인** | 👁 시선이탈 / 🙈 얼굴미감지 / 🖥 화면이탈 / 🎙 음성경고 카테고리 분류 |
| **감독 기준 설정** | 시선 이탈 허용 시간, 최대 경고 횟수 동적 설정 |
| **보안** | Groq API 키 서버사이드 보관 (브라우저 미노출) |

## 🏗️ 기술 스택

### Frontend
- Vanilla JS (ES Modules)
- MediaPipe FaceMesh — 홍채 기반 시선 추적
- Web Speech API — 실시간 음성 인식
- KaTeX — LaTeX 수식 렌더링
- PDF.js — PDF → 이미지 변환

### Backend
- **FastAPI** (Python) — REST API
- **MySQL** (Railway) — 데이터 저장
- **JWT** — 인증/인가
- **Groq API** (LLaMA 4 Scout Vision / LLaMA 3.1) — AI 문제 생성, 음성 분석, 면담 챗봇

### 인프라
- **Railway** — 백엔드 + DB 호스팅
- **Vercel** — 프론트엔드 정적 호스팅

## 🚀 사용 방법

### 관리자

#### 1. 로그인
`관리자 대시보드` 접속 → 아이디/비밀번호 입력

#### 2. 설정
`설정` 메뉴 → Groq API 키 입력 → 감독 기준(시선 이탈 허용 초, 최대 경고 횟수) 설정 → 저장

#### 3. 시험 생성
`시험 생성` 메뉴 → 시험명·시간 입력 → PDF 업로드 → AI 문제 생성 → 검토 후 저장 → 시험 활성화

#### 4. 실시간 감독
`실시간 감독` 메뉴 → 응시 중인 학생 카드 확인 → `관리자 개입` 버튼으로 메시지/일시정지/강제종료

#### 5. 결과 확인
`응시 결과` 메뉴 → 시험 선택 → 응시자별 상세 확인
- **답안 탭**: 문항별 정오답 확인
- **이벤트 탭**: 카테고리별 요약 + 타임라인
- **음성기록 탭**: 시험 중 발화 전체 기록
- **AI면담 탭**: 면담 대화 내역

---

### 응시자

#### 1. 접속
`응시자 페이지` 접속 → 학번 / 이름 입력 → 시험 선택 → 시험 시작

#### 2. 응시 환경
- 카메라와 마이크 권한 허용 필수
- 시험 중 다른 탭/창 전환 금지
- 카메라에 얼굴이 항상 보이도록 유지

#### 3. 경고 시스템
```
경고 2회 → AI 감독관 면담
경고 4회 → AI 감독관 면담 (2차)
경고 6회 → 0점 강제 종료
```
